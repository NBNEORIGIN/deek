# Rolf — Refine Color 6090 UV flatbed

Rolf is a Refine Color 6090 UV flatbed printer. It runs twin Epson
i1600 heads (CMYK + white) on rigid substrates and is one of NBNE's
two main UV flatbeds alongside the Mimaki. The varnish head is
present but blocked / unused. Earlier internal docs incorrectly
described Rolf as a Mimaki UV or as a Roland — both wrong; the
machine has always been the Refine Color 6090.

## Issue Description

This is considered unusual behaviour according to Graphtec GB technical support, as maintenance prompts should clear automatically once the required maintenance is performed.

## Troubleshooting Protocol

### Step 1: Daily Maintenance Verification

When a maintenance prompt appears:
1. Perform the daily maintenance procedure as normal
2. Check the Mutoh Status Monitor software for current machine status
3. Attempt to access the printer menu to verify system responsiveness

**Note:** NBNE staff (Gabby) confirmed daily maintenance was completed but the prompt persisted.

### Step 2: Power Cycling Procedure

If the maintenance prompt remains after completing maintenance:

1. Power down the machine completely
2. Power up the machine
3. Check if the prompt has cleared

**Critical Finding:** Initial power cycling did not resolve the issue. NBNE required a **second power cycle** before the prompt finally cleared from the display.

## Best Practices and Recommendations

### For Persistent Maintenance Prompts:

1. Always complete the indicated maintenance first
2. Check Mutoh Status Monitor software
3. Power cycle once and wait
4. If prompt persists, power cycle a second time
5. Document with photos (NBNE sent display images to support)
6. Contact Graphtec GB support if issue continues after multiple attempts

### For Service Booking:

1. Request service via support@graphtecgb.co.uk
2. Provide machine location immediately in initial request

## LED Module Systems

### FL Micron Modules

Standard modular LED system for backlit installations. Key specifications:

- **Halo illumination**: 120 × FL Micron modules (warm white) quoted for 1039mm × 822mm × 30mm panel
- **Driver pairing**: 60W 12V driver suitable for typical installations
- **Mounting requirement**: Minimum 30mm standoff locators required for proper light distribution
- **Cost example**: £72 + VAT (120 modules + 60W driver for 1039mm × 822mm panel)

### Cool/Warm White Module Selection

- **Cool white**: Standard for general signage, mixed-colour graphics
- **Warm white**: Specified for interior panels with consistent tone requirements
- **Perimeter mounting**: For tray signs, modules fitted around returns on inside face

## Tray Sign Illumination

### Typical Configuration (ACM Tray Signs)

Based on correspondence for ~4.5m perimeter tray sign:

1. **Construction**: ACM tray with 30mm returns
2. **Diffusion**: 3mm opal acrylic panel fitted to inside face, with translucent vinyl applied
3. **Module placement**: Cool white modules mounted around perimeter on inside returns
4. **Critical limitation**: Edge illumination creates spotting at 30mm return depth — back illumination also problematic at shallow returns

### Design Considerations

Amy Trow (First Lite LED) flagged that:
- 30mm returns are too shallow for even light distribution with either edge or back illumination
- Testing required before committing to module specification
- Alternative approaches needed for shallow return depths

## LED Neon Flex Systems

### Infinity Neon

Used successfully on Bamburgh Castle project and similar applications:

- **Application**: Channel routing in foam or similar substrate
- **Tube configuration**: Clear tube with reel-type LED insert
- **Colour options**: Cool white standard for backlit multi-colour graphics
- **Path length example**: 1381mm path quoted for circular backlit sign
- **Driver**: 48W 12V suitable for typical installations

### Typical Quote Structure

Quote for interior backlit circular sign (path length 1381mm):
- 4 × Clear tube
- 4 × Reel cool white
- 1 × 48W 12V driver
- 1 × Power LED
- **Total**: £37.00 + VAT

### Installation Method

For backlit acrylic with machined foam returns:
1. Machine groove into foam return to hold neon strip
2. Route neon through groove along design path
3. Front acrylic panel applied over illuminated assembly
4. Interior use only (unless specifically IP-rated system specified)

## LED Reel/Strip Systems

### Standard Reels

- **5mm wide reels**: Only option for very small letter forms or tight spaces
- **Warning**: Standard reels are NOT S-shaped, making installation in channels difficult
- **Weather exposure**: Reels perform poorly in rain unless fully sealed
- **Application limit**: Letters/channels must have adequate depth — many small letters too small for any module system

### Limitations on Small Letterforms

Amy Trow advised that letters below certain size thresholds are "too small for any module to fit in" — LED reel may be only option, but installation challenging and weather protection critical.

## Key Technical Limitations

1. **30mm return depth insufficient** for even illumination with standard modules
2. **Small letterforms** exclude most modular systems — reels only option
3. **External/rain exposure** incompatible with standard LED reels unless fully sealed
4. **Fret-cut designs** compatible with neon flex systems (confirmed non-issue)
5. **Standoff mounting** requires minimum 30mm locators for backlit panels

## Best Practices

1. **Consult Amy/Sophie at First Lite LED** early in design phase with dimensions, return depths, and application details
2. **Test shallow returns** before committing to module specification
3. **Specify interior/exterior** and weather exposure requirements upfront
4. **Calculate perimeter/path length** accurately for module quantity estimation
5. **Consider neon flex** for complex paths, channel letters, and custom shapes where modules impractical

### Module Selection Guidance

**Return Depth Requirements**:
- **Minimum 60mm returns recommended** for single-sided signs using edge modules
- **30mm returns are insufficient** — will cause visible spotting with both edge and back illumination
- For shallow return depths, the tray must be recut to accommodate proper module spacing

## Technology Notes: CSP vs COB vs SMD

Based on supplier marketing materials (unverified):

**CSP (Chip-Scale Package)**:
- Compact design with mini LEDs and resistors
- Better bend resistance than COB
- No yellowing effect
- Wider operating temperature range

**COB (Chip-on-Board)**:
- Can show yellowish tint with profiles
- More stray light than CSP
- Less flexible

**SMD (Surface-Mount Device)**:
- Traditional technology
- Less durable than CSP (per manufacturer claims)

**NBNE Position**: Continue using First Lite LED modules for standard applications. New technologies require testing before specification.

## Procurement Guidelines

1. **Default to First Lite LED** for all standard sign module requirements
2. **Always specify return depth** when requesting quotes — 60mm minimum for edge modules
3. **Request warm/cool white specification** upfront
4. **Calculate perimeter** before ordering to determine module quantity
5. **International suppliers**: Require samples and testing before approval — do not rely on marketing claims

---

## Known Issues & Lessons

- **Shallow returns cause spotting**: 30mm insufficient for even illumination
- **Driver sizing**: Match to total module count (60W suitable for ~120 modules)
- **Mounting clearance**: Budget for 30mm locators minimum for backlit panels
- **Acrylic thickness**: 3mm opal acrylic confirmed suitable with translucent vinyl overlay

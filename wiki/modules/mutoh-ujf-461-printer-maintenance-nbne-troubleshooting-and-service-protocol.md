## Common Issue: Persistent Maintenance Prompt

### Problem Description

In March 2026, NBNE experienced an issue where the Mutoh UJF 461 displayed a maintenance prompt that remained on screen even after completing the required daily maintenance procedures. This prevented normal operation.

### Troubleshooting Steps (In Order)

1. **Perform Daily Maintenance**
   - Complete all standard daily maintenance procedures as per manufacturer guidelines
   - Check if prompt clears automatically

2. **Power Cycle the Machine**
   - Power down the printer completely
   - Wait a few moments
   - Power up the machine
   - Check if the maintenance prompt has cleared

3. **Repeat Power Cycle if Necessary**
   - If the first power cycle doesn't resolve the issue, attempt a second power cycle
   - According to Graphtec GB: "Unusual for this prompt to stay on the machine if the maintenance has been carried out"

### NBNE Resolution (March 2026)

The issue was resolved after performing daily maintenance followed by **two separate power cycles**. Gabby Bassett reported on 10 March 2026: "Just to update, the prompt has gone from the display now. We powered down again and it appears to have cleared it."

### Key Takeaway

If a maintenance prompt persists after completing maintenance:
- Do not immediately escalate to technical support
- Attempt power cycling at least twice before contacting supplier
- Document the issue and resolution steps taken

## Maintenance Diagnostics

### Pre-Support Checks

Before contacting technical support, gather the following information:
- Has daily maintenance been completed?
- What is displayed on the printer screen? (Take photo)
- Can you access the menu system?
- What status is shown in Mutoh Status Monitor software?
- Has power cycling been attempted?

**LED Reels/Strip**
- 5mm wide reels available for compact applications
- **Warning**: Standard LED reels not suitable for rain exposure unless fully sealed
- Amy recommends testing edge illumination before ordering for shallow return applications

### Technical Guidance from First Lite LED

**Spotting Issues on Shallow Returns**
- 30mm returns will likely show spotting with both edge illumination and back illumination
- Always test edge modules before committing to full order on shallow applications
- Single-sided signs with 30mm returns particularly problematic for even illumination

**Application-Specific Recommendations**
- Backlit ACM trays: Fit opal acrylic on inside face, mount modules around return perimeter on inside
- Halo effects: Modules fitted to perimeter
- Small letter applications: Standard modules too large; 5mm reels may work but difficult to fit in 'S' shapes

## NBNE Standard Practices

### Typical Build Specifications
- **Backlit panels**: Opal acrylic diffuser with translucent vinyl
- **Tray signs**: ACM tray construction with opal acrylic fitted to inside face
- **Standoff mounting**: 10mm standoff mounts standard for single-piece ACM panels
- **SimpleFIX fixings**: Used for illuminated standoff signs with LED strip behind clear acrylic elements

### Colour Temperature Selection
- **Warm white**: Standard for halo-lit panels and commercial applications
- **Cool white**: Used for backlit tray signs and specific client requirements
- Always specify colour temperature when ordering

## Procurement Process

1. Contact Amy@firstliteled.co.uk with project specifications
2. Provide dimensions, application type, and mounting details
3. Specify colour temperature (warm/cool white)
4. Amy will quote specific module counts and driver requirements
5. Allow for testing on shallow return or complex applications before full order

## Key Warnings

- **Rain exposure**: Standard LED reels unsuitable unless fully sealed
- **Shallow returns**: 30mm or less will likely show spotting - test before ordering
- **Small letters**: Most modules too large - consider 5mm reels but note installation difficulty
- **Locator requirements**: Minimum 30mm standoffs required for proper module function
